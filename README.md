# SentriGOV2

SentriGOV2 is a real time network monitoring and intrusion detection platform focused on visibility, simplicity, and performance. It helps users monitor network activity, understand traffic behavior, and detect suspicious patterns in an accessible and meaningful way.

The platform is built to be usable in real environments while remaining easy to understand and extend.
